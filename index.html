Setup

How to Install

Requires a Nintendo 3DS with Luma3DS CFW and the Rosalina menu plugin loader.

1

Download the latest release

Grab the .zip from the Releases page and extract its contents.

2

Copy the .3gx file to your SD card

Place the plugin file in /luma/plugins/ folder on your 3DS SD card. Create the folder if it doesn't exist.

3

Launch your Pokémon game

Boot your Gen 6 game normally. Luma3DS will automatically load the plugin on startup.

4

Open the plugin menu

Press the button combo configured in Rosalina (usually L+Down+Select) to open the plugin menu in-game.

5

Read the built-in guide

Head to App Guide inside the plugin — 20 pages of beginner-friendly instructions are waiting for you.
