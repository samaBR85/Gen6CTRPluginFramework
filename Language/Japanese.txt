NOTE_PSS_O_POWER_CENTER :: Oパワーセンター - 全18種のOパワーとチャージゲージを一覧表示するボードです。\n\n上画面:\n- D-Pad: パワーを選ぶ\n- L / R: レベルを変える（Aで切り替え）\n- Y: 生データ表示（上級者向け）\n\nボタン:\n- すべて最大: 全Oパワーを最大レベルに\n- 解放: 全Oパワーをレベル1に\n- ロック: すべて再ロック（レベル0に戻す）\n- 補充: 今すぐゲージを満タンに（いつでも押せます）\n- 維持: ゲージを常に満タンに保つ（チャージ待ちなし）\n\n変更を保持するにはゲームをセーブしてください。
OPC_TITLE :: Oパワーセンター

OPC_NAMES :: こうげきのOパワー :: ぼうぎょのOパワー :: とくこうのOパワー :: とくぼうのOパワー :: すばやさのOパワー :: めいちゅうのOパワー :: きゅうしょのOパワー :: HPかいふくのOパワー :: PPかいふくのOパワー :: であいのOパワー :: しのびあしのOパワー :: けいけんちのOパワー :: しょうきんのOパワー :: つかまえやすさのOパワー :: タマゴふ化のOパワー :: おねだりのOパワー :: なかよしのOパワー :: かんぜんかいふくのOパワー

REFILL_DONE :: 手持ちを回復しました！

MENU_HEALTH_MANA :: HPとPP
NOTE_HEALTH_MANA :: 手持ちのHPとPPを回復・固定します。\n\n常に最大（切り替え）: 戦闘中HP/PPを満タンに固定 - ひんしにならず、PP切れもなし。\n即時補充（切り替え）: ひとつONにするとショートカットが有効になり、押すと一度だけ補充します（初期はOFF - 任意設定。OFFでボタンが解放されます）。HP補充はフィールドで動作し、戦闘中はむげんのHPを使ってください。
HM_HDR_ALWAYS_MAX :: 常に最大
HM_HDR_BIND :: 即時補充 / ボタン操作
INFINITE_HEALTH :: むげんのHP (HP)
NOTE_INFINITE_HEALTH :: 戦闘中これをONにすると手持ちのHPを最大に保ちます - 次の攻撃でバーが満タンに戻り、ひんしになりません。
INFINITE_MANA :: むげんのPP (PP)
NOTE_INFINITE_MANA :: 戦闘中これをONにすると全わざのPPを最大に保ちます - わざが切れません。
REFILL_HEALTH :: HPを補充 (HP)
NOTE_REFILL_HEALTH :: 手持ち全員のHPを補充します。\n\n- ON: ショートカットを有効化（初期はZL）。\n- OFF: ボタンを解放。\n\nフィールド: ショートカットを押すと手持ち全員が完全回復。\n戦闘中: リアルタイムのみ - 攻撃を受けたとき、HPバーのアニメーション中にショートカットを押すとバーが減らずに満タンになります。常に満タンにするにはむげんのHPを使ってください。\n\nショートカットメニューで再設定できます。
REFILL_MANA :: PPを補充 (PP)
NOTE_REFILL_MANA :: 手持ち全員のPPを補充します。\n\n- ON: ショートカットを有効化（初期はZR）。\n- ショートカットを押す: 今すぐ補充。\n- OFF: ボタンを解放。\n\n戦闘中もフィールドでも動作します。\n\nショートカットメニューで再設定できます。
HOTKEY_REFILL_HEALTH :: HPを補充

NOTE_DEXNAV_MAX_SEARCH :: 全721種のDexNavサーチレベルを最大にします（ORASのみ）。\n\nサーチレベルが高いほど、DexNavで色違い・隠れ特性・タマゴわざ・レアな隠れポケモンの出現率が上がります。\n\n一度きり: 選ぶと適用され、ゲームに保存されます。\n\n警告: これは元に戻せません。
FAV_DEXNAV_MAX_SEARCH :: DexNav最大
DEXNAV_MAX_DONE :: DexNavサーチレベルを最大にしました！
DEXNAV_CONFIRM_BODY :: 全721種のDexNav\nサーチレベルを\n最大にします。\nゲームに保存され\n元に戻せません。
